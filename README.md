# Sentiment Analysis Project

This project is a simple sentiment analysis system developed using Python and machine learning techniques. It is designed to classify a given piece of text as either positive or negative. The model has been trained on a labeled dataset using TF-IDF vectorization to convert textual data into numerical features, and a machine learning algorithm to perform the classification. The project also includes a user-friendly interface built using Streamlit, allowing users to interact with the model in real time.

The application uses several important technologies and libraries, including Python as the programming language, Pandas for data handling, Scikit-learn for building and training the machine learning model, and Streamlit for creating the web-based interface. The trained model and vectorizer are saved as `.pkl` files and are loaded during runtime to make predictions efficiently without retraining.

To run this project, first ensure that Python is installed on your system. Then open the project folder in a code editor like VS Code and open a terminal in the project directory. Install the required libraries by running the command: `pip install streamlit pandas scikit-learn`. After installing the dependencies, navigate to the main project folder and run the application using the command `python -m streamlit run app.py`. This will start a local server, and the application can be accessed in a web browser at `http://localhost:8501`.

Once the application is running, users can enter any text into the input field and click the analyze button to view the predicted sentiment. The system demonstrates the complete workflow of a machine learning project, including data preprocessing, model training, saving and loading models, and deploying them through an interactive interface. This project is intended for educational purposes and provides a clear understanding of how natural language processing and machine learning can be applied to real-world problems.
